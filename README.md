## 5. ‡∏´‡∏•‡∏±‡∏Å‡∏Å‡∏≤‡∏£‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì‡∏ó‡∏≤‡∏á‡∏™‡∏ñ‡∏¥‡∏ï‡∏¥ (Statistical Methods)

### 5.1 ‡∏ú‡∏•‡∏ï‡πà‡∏≤‡∏á‡∏Ñ‡πà‡∏≤‡πÄ‡∏â‡∏•‡∏µ‡πà‡∏¢ (Mean Difference)

[
\text{Mean Difference} = \bar{X}_1 - \bar{X}_2

---

### 5.2 ‡∏™‡πà‡∏ß‡∏ô‡∏Ñ‡∏•‡∏≤‡∏î‡πÄ‡∏Ñ‡∏•‡∏∑‡πà‡∏≠‡∏ô‡∏°‡∏≤‡∏ï‡∏£‡∏ê‡∏≤‡∏ô (Standard Error; SE)

[

]

---

### 5.3 ‡∏Å‡∏≤‡∏£‡πÄ‡∏•‡∏∑‡∏≠‡∏Å‡∏Ñ‡πà‡∏≤ Z ‡∏´‡∏£‡∏∑‡∏≠ t ‡∏≠‡∏±‡∏ï‡πÇ‡∏ô‡∏°‡∏±‡∏ï‡∏¥

* ‡∏´‡∏≤‡∏Å‡∏Ç‡∏ô‡∏≤‡∏î‡∏ï‡∏±‡∏ß‡∏≠‡∏¢‡πà‡∏≤‡∏á‡πÅ‡∏ï‡πà‡∏•‡∏∞‡∏Å‡∏•‡∏∏‡πà‡∏° ‚â• 30 ‚Üí ‡πÉ‡∏ä‡πâ‡∏Ñ‡πà‡∏≤ Z = 1.96
* ‡∏´‡∏≤‡∏Å‡∏Ç‡∏ô‡∏≤‡∏î‡∏ï‡∏±‡∏ß‡∏≠‡∏¢‡πà‡∏≤‡∏á < 30 ‚Üí ‡πÉ‡∏ä‡πâ‡∏Ñ‡πà‡∏≤ t ‡∏à‡∏≤‡∏Å t-distribution ‡∏ó‡∏µ‡πà‡∏£‡∏∞‡∏î‡∏±‡∏ö 95% (Œ± = 0.05) ‡πÇ‡∏î‡∏¢‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì‡∏à‡∏≤‡∏Å degrees of freedom (df)

[

---

### 5.4 ‡∏ä‡πà‡∏ß‡∏á‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏ä‡∏∑‡πà‡∏≠‡∏°‡∏±‡πà‡∏ô 95% (95% Confidence Interval)

[
95%,CI = (\bar{X}_1 - \bar{X}_2) \pm c \times SE
]
‡πÇ‡∏î‡∏¢‡∏ó‡∏µ‡πà

* (c = 1.96) (‡∏Å‡∏£‡∏ì‡∏µ Z)
* ‡∏´‡∏£‡∏∑‡∏≠ (c = t_{0.975, df}) (‡∏Å‡∏£‡∏ì‡∏µ t-distribution)

---

### 5.5 ‡∏Å‡∏≤‡∏£‡∏ó‡∏î‡∏™‡∏≠‡∏ö‡∏ó‡∏≤‡∏á‡∏™‡∏ñ‡∏¥‡∏ï‡∏¥ (t-statistic)

[

---

### 5.6 ‡∏Ç‡∏ô‡∏≤‡∏î‡∏≠‡∏¥‡∏ó‡∏ò‡∏¥‡∏û‡∏• (Effect Size)

#### 5.6.1 Cohen‚Äôs d

[
d = \frac{\bar{X}_1 - \bar{X}*2}{SD*{pooled}}
]

[
SD_{pooled} = \sqrt{\frac{(n_1-1)SD_1^2 + (n_2-1)SD_2^2}{n_1+n_2-2}}
]

#### 5.6.2 Hedges‚Äô g (Bias-corrected Effect Size)

[
g = d \times \left(1 - \frac{3}{4df - 1}\right)
]
